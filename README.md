# 🏠 Find My Nest

Find My Nest is an AI-powered Rental & PG Management Platform designed to simplify accommodation discovery and property management in Tiptur. The platform connects tenants and property owners through a centralized digital ecosystem, making the rental process transparent, efficient, and accessible.

## 🚀 Overview

Finding rental houses and PG accommodations in smaller towns is often dependent on brokers, word-of-mouth recommendations, and unorganized listings. Find My Nest addresses this challenge by providing a modern web platform where:

* Tenants can discover verified rental properties and PGs.
* Property owners can list and manage accommodations.
* Users can send enquiries directly to owners.
* An AI-powered assistant helps users with property-related queries.

## ✨ Features

### For Tenants

* Browse rental houses and PG accommodations
* Search and filter by location, rent, and property type
* View detailed property information
* Send enquiries directly to owners
* AI-powered property assistance

### For Property Owners

## 🎯 Problem Statement

In towns like Tiptur, finding rental accommodations is often time-consuming and fragmented. Property owners struggle to manage listings and enquiries, while tenants lack a centralized platform for discovering available properties.

Find My Nest bridges this gap through a user-friendly digital solution that simplifies property discovery and management.
